# 데이터 안전성 답안

## 개요
- 앱에서 사용자 데이터를 수집하거나 공유합니까? **예** (광고 SDK를 통해)
- 앱에서 사용자 데이터를 다른 회사나 조직과 공유합니까? **예** (광고 목적)

## 데이터 유형별 답변

### 위치
- 대략적인 위치: **수집 안 함**
- 정확한 위치: **수집 안 함**

### 개인 정보
- 이름: **수집 안 함**
- 이메일 주소: **수집 안 함**
- 사용자 ID: **수집 안 함**
- 주소: **수집 안 함**
- 전화번호: **수집 안 함**

### 금융 정보
- 수집 안 함

### 건강 및 피트니스
- 수집 안 함

### 메시지
- 수집 안 함

### 사진 및 동영상
- 수집 안 함

### 오디오
- 수집 안 함

### 파일 및 문서
- 수집 안 함

### 캘린더
- 수집 안 함

### 연락처
- 수집 안 함

### 앱 활동
- 수집 안 함 (직접)

### 웹 브라우징
- 수집 안 함

### 앱 정보 및 성능
- 비정상 종료 로그: **수집 안 함** (직접 수집 없음)

### 기기 또는 기타 ID
- **기기 또는 기타 ID: 수집함**
  - 수집 목적: **광고**
  - 데이터 공유 여부: **예** (Google AdMob을 통해)
  - 필수 여부: **필수** (광고 표시에 필요)
  - 사용자가 삭제 요청 가능: **아니오** (Google 광고 설정에서 관리)

## 보안 관행
- 데이터가 전송 중에 암호화됩니까? **예** (HTTPS)
- 사용자가 데이터 삭제를 요청할 수 있습니까? **해당 없음** (직접 수집하는 개인 데이터 없음)

## 참고
- AdMob SDK가 광고 ID(GAID)를 수집하며, 이는 광고 개인화 목적
- 앱 자체는 생년월일 등 입력 데이터를 서버로 전송하지 않음 (기기 내 처리)
